# Матрица совместимости методов

Это method-level контракт по backend compatibility, routing mode, fallback
behavior и semantic risk. Документ генерируется из
[capabilities.yaml](capabilities.yaml).

Он фокусируется на routed, downgraded, bridged и risk-bearing RPC families, а
не на каждом пересекающемся Apache HMS методе. Front-door-local security token
API по-прежнему описаны в [README.ru.md](README.ru.md).

Легенда статусов:

- `supported`: нормальный путь для этой backend/runtime комбинации
- `degraded`: запрос работает через downgrade, default-catalog pin или compatibility response
- `rejected`: запрос падает явно вместо угадывания маршрута
- `unsafe`: запрос может маршрутизироваться, но multi-catalog semantics не стоит считать production-safe без отдельной валидации

Обновить сгенерированную таблицу можно так:

```bash
mvn -o -q -Dtest=CapabilityMatrixDocSyncTest -Dcapabilities.updateReadme=true test
```
